# ToDo App using React.js/Vite with latest Version React-router-dom / Redux- Toolkit

## Get starting
     npm i 
     npm run dev

## Posibilities

    add new task
    delete a task
    edit a task
    filter a tasks
    etc...

## Including

    react router dom v6.14.2
    react-redux
    @reduxjs/toolkit
    uuid
    framer-motion
    react-icons
    match-sorter
    sass
    sort-by
    localforage
